必須

　Node.js：LTS 推奨（v20 以上）

　インストール後に node -v と npm -v が通ること


推奨（エディタ）

　VS Code

　拡張機能：Vue - Official (Volar)、Prettier

手順

　1.vue_templateフォルダをvscodeで開く

　2.ターミナルでカレントディレクトリをvue-projectにする
　cd vue-project

　3.ターミナルでnpm iを実行し、パッケージをインストールする
　npm i

使用技術
　フロントエンドフレームワーク
　Vue.js

　ビルドツール
　Vite

　CSSフレームワーク
　Vutify.js

　プログラミング言語
　TypeScript

　コード品質維持
　ESLint
